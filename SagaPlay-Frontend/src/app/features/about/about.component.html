This website was created using front-end and back-end technologies.

The front-end technologies used are as follows:

  • Angular 17 (With Standalone Components)
  • HTML / CSS
  • Bootstrap 5
  • Auth0 (For user authentication)


The back-end technologies used are as follows:

  • Microsoft .NET 8 Microservices (REST API)
  • RabbitMQ (For inter service communication)
  • Ocelot (For the API Gateway)
  • Entity Framework Core 8
  • PostGres SQL Server
